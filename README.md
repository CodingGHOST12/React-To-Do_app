# React To-Do List Application

A simple, interactive To-Do List application built using React and Vite. Users can add, edit, delete, and complete tasks.

## How to Run the Application Locally

1. **Clone the repository:**
   git clone <YOUR_GITHUB_REPO_URL_HERE>

2. **Navigate into the project directory:**
   cd react-todo-app

3. **Install the dependencies:**
   npm install

4. **Start the development server:**
   npm run dev

5. Open your browser and visit the local link provided in the terminal (usually http://localhost:5173).
